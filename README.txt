COMANDOS:
dir: Muestra los contenido del directorio actual
dir "argumento": Muestra el contenido del directorio "argumento"
cd "argumento": Ingresa al directorio "argumento"
pause: Detiene la ejecucion del shell hasta que el usuario pulse "ENTER"
clr: Limpia el terminal
quit: Sale del shell, termina el proceso
echo "comentario": Imprime en pantalla el "comentario"
